UI ENHANCEMENT REQUEST – GIGZITO CATEGORY CAROUSEL

We are expanding the Gigzito platform categories and need to redesign the current bottom navigation menu into a horizontal carousel category selector.

CURRENT ISSUE
The current UI uses a fixed bottom navigation with limited slots:
Feed | Music | Marketing | Crypto | Courses | Profile

This will not scale because Gigzito will continue adding more verticals.

NEW DESIGN REQUIREMENT
Replace the fixed category buttons with a horizontally scrollable carousel category menu positioned directly below the Gigzito logo and above the video feed.

LAYOUT STRUCTURE

Gigzito Logo
Horizontal Category Carousel
Main Video Feed
Bottom Navigation (Feed + Profile only)

CATEGORY CAROUSEL BEHAVIOR

• Horizontal swipe left/right
• Smooth scroll animation
• Tap a category to filter the feed
• Selected category highlights in red
• Other categories appear in neutral color
• Carousel should allow unlimited categories

INTERACTION

When user taps a category:

This redesign allows Gigzito to support many verticals while keeping the UI clean and mobile friendly.
