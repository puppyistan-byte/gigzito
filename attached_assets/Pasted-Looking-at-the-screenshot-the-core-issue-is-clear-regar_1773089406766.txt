What changed in this fix:

The old code had two nested divs — an outer flex container and an inner flex container. Both used height: 100%, meaning the inner div's height depended on the outer div's height, which depended on the feed-item. On VPS, that chain was silently collapsing, causing all the position: absolute children to anchor to a zero-height box and render in document flow instead.

This version uses one single div with explicit height: 100%; position: relative; overflow: hidden. There's no chain — just one container inheriting height from the feed-item directly, then all children absolutely positioned inside it.
