GZFlash Sales Module — Mobile API & Design Spec
Base URL
https://gigzito.com (production VPS) / http://localhost:5000 (dev)

Authentication
All write endpoints require a session cookie from /api/auth/login + /api/auth/mfa/verify. The public GET is cookie-free.

POST /api/auth/login
Body: { "email": "...", "password": "..." }
Response: { "mfaRequired": true, "email": "...", "devCode": "123456" }
POST /api/auth/mfa/verify
Body: { "email": "...", "code": "123456" }
Response: { user: { id, role, subscriptionTier }, profile: { username, avatarUrl, ... } }
Endpoints

Returns all active, non-expired ads sorted by potency score descending.

2. GET /api/gz-flash/mine — Owner's ads (auth + GZBusiness tier)
Returns only the logged-in user's ads (all statuses).

3. POST /api/gz-flash — Create ad (auth + GZBusiness tier)
// Request body
{
  "title": "Summer Blowout",           // string, required, min 1 char
  "artworkUrl": "https://...",          // string | null
  "retailPriceCents": 9999,            // integer cents, e.g. $99.99 = 9999
  "discountPercent": 75,               // integer 1–99
  "quantity": 10,                      // integer ≥ 1
  "durationMinutes": 120,              // integer 5–43200 (max 30 days)
  "displayMode": "countdown"           // "countdown" | "slots"
}
// Response 201 — full ad object (same shape as GET)
4. PUT /api/gz-flash/:id — Update ad (auth + GZBusiness tier, must own ad)
Same body shape as POST.

5. DELETE /api/gz-flash/:id — Delete ad (auth, must own ad)
// Response
{ "ok": true }
6. POST /api/gz-flash/:id/claim — Claim a deal (no auth required)
Decrements available slots. Fails if claimedCount >= quantity or ad is expired/paused.

// Response — updated ad object
// Error (400): { "message": "No slots remaining" }
Admin-Only Endpoints (ADMIN / SUPER_ADMIN / SUPERUSER role)
Method	Path	Description
GET	/api/admin/gz-flash	All ads, all statuses, with owner email
PATCH	/api/admin/gz-flash/:id/status	Body: { "status": "active" | "paused" | "expired" }
DELETE	/api/admin/gz-flash/:id	Hard delete any ad
POST	/api/admin/gz-flash/:id/message	Body: { "note": "..." } — sets adminNote on the ad
Score Formula (compute client-side for live preview)
COMFORT = 100  // $100 impulse-buy threshold
savingsIndex   = (retailDollars - flashDollars) / retailDollars
timeFactor     = 1 + (1 - timeRemainingMinutes / durationMinutes)    // 1.0 → 2.0
scarcityFactor = 1 + (1 - slotsRemaining / quantity)                 // 1.0 → 2.0
priceFriction  = 1 / (1 + flashDollars / COMFORT)                    // lower price = higher
score = min(100,  savingsIndex × timeFactor × scarcityFactor × priceFriction × 100)
The server recalculates potencyScore every 60 seconds automatically. At launch: timeFactor = 1.0, scarcityFactor = 1.0. Both climb toward 2.0× as time and slots drain.

displayMode — Two Card Variants
"countdown" — Show a live countdown timer (expiresAt - now). Drives time urgency.
"slots" — Show quantity - claimedCount slots remaining. Drives scarcity urgency.
Price Conversion
All prices are stored and transmitted in cents (integers).

Display: divide by 100 → $95.95
Send to API: multiply by 100 → 9595
