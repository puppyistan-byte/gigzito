Mobile Profile API
All endpoints use Authorization: Bearer <token> (same token from /api/mobile/login + /api/mobile/mfa/verify). Works for any subscription tier.

GET /api/mobile/me — Fetch full profile
Returns the user's account info and their complete GeeZee profile.

Authorization: Bearer <token>

PUT /api/mobile/profile — Edit GeeZee profile
Send any subset of profile fields — only what you include gets updated.

PUT /api/mobile/profile
Authorization: Bearer <token>
Content-Type: application/json
Body (all fields optional — send only what changed):

{
  "displayName": "My Name",
  "bio": "Updated bio text",
  "username": "myhandle",
  "avatarUrl": "https://...",
  "thumbUrl": "https://...",
  "primaryCategory": "MARKETING",
  "location": "Portland OR",
  "contactEmail": "me@example.com",
  "contactPhone": "+1-503-555-0100",
  "contactTelegram": "@myhandle",
  "websiteUrl": "https://mysite.com",
  "instagramUrl": "https://instagram.com/me",
  "youtubeUrl": "https://youtube.com/@me",
  "tiktokUrl": "https://tiktok.com/@me",
  "facebookUrl": "https://facebook.com/me",
  "discordUrl": "https://discord.gg/...",
  "twitterUrl": "https://twitter.com/me",
  "photo1Url": "https://...",
  "photo2Url": "https://...",
  "photo3Url": "https://...",
  "photo4Url": "https://...",
  "photo5Url": "https://...",
  "photo6Url": "https://...",
  "showPhone": false
}
Response: { "ok": true, "profile": { ...full profile } }

PATCH /api/mobile/account — Change email address
PATCH /api/mobile/account
Authorization: Bearer <token>
Content-Type: application/json
{ "email": "newemail@example.com" }
Response: { "ok": true, "account": { id, email, role, subscriptionTier } }

Errors:

400 — missing or invalid email format
409 — email already in use by another account
401 — bad/expired token
Note: Image uploads (avatar, photos) still go through POST /api/upload/image with a Bearer token and multipart/form-data — that endpoint already supports Bearer tokens. Returns { "url": "/uploads/filename.jpg" } which you can then save via PUT /api/mobile/profile.
